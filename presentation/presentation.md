## 조작법

- **스페이스바 / → / ↓** : 다음 (서브스텝 있으면 스텝 먼저)
- **← / ↑** : 이전 (서브스텝 역순 되돌리기)
- **화면 우측 클릭** : 다음
- **화면 좌측 클릭** : 이전
- **Home** : 첫 슬라이드
- **End** : 마지막 슬라이드

## 서브스텝 시스템

슬라이드 내에서 `.step` 클래스를 가진 요소는 자동 표시되지 않고, 스페이스바를 누를 때마다 하나씩 페이드인됩니다. 모든 스텝이 표시된 후 다음 스페이스바에서 다음 슬라이드로 이동합니다.

- CSS에서 `.step` 요소에 `animation: none !important` 필수
- 초기 스타일: `opacity:0; transform:translateY(24px);`
- 슬라이드를 떠나면 자동 리셋

### S4 서브스텝 흐름
1. 진입: "한국 사회는 갈등으로 가득하다 (74%)" 화면 중앙
2. 스텝1: 텍스트 축소+위 이동 → 화살표↓ + "그 갈등은 필연적으로 논쟁을 일으킨다"
3. 스텝2~4: 카드 순차 등장

## 새 슬라이드 추가 방법

1. `css/slideN.css` 생성 (기존 파일 복사 후 `#sN`으로 치환)
2. `index.html`에 `<link rel="stylesheet" href="css/slideN.css">` 추가
3. `index.html`에 `<section class="slide" id="sN">` 추가
4. 서브스텝 필요 시 요소에 `class="step"` + `style="opacity:0; transform:translateY(24px);"` 추가

## 디자인 통일 규칙

- 헤더: `align-items:center; gap:1rem; padding-bottom:1.2rem; margin-bottom:clamp(2rem,5vh,3.5rem)`
- 페이지 번호: `.page-num` 뱃지 (Cinzel, 0.7rem, 골드 테두리)
- 타이틀: `clamp(1.4rem, 3.5vw, 2.2rem)` 골드 그라데이션
- 푸터: `.s-footer-abs` (absolute bottom, 골드 경계선)
- 배경: 공용 bg-gradient + scanline + vignette + left-bar + bottom-bar

## 출처 데이터

### S3 제안 배경 (국민통합위원회 2025, 한국갤럽, n=7,000)
- 5대 사회갈등 심각 인식 평균: 74%
  - 진보·보수 92.4%, 부유·서민 77.3%, 기성·청년 71.8%, 수도·지방 69.5%, 남성·여성 61.0%
- 부정적 감정: 81% (분노 27%, 혐오 22%, 슬픔 16%, 기타 부정 16%)
- 다른 의견과 대화할 의향: 70%

### S4 원인 분석 (작업 중)
- "한국 사회는 갈등으로 가득하다" (74%)
- "그 갈등은 필연적으로 논쟁을 일으킨다"

### S5 해결 전략
- AI 60% + 시민 40% 복합 판결
- 판결 플로우: 입론 → 반론 → AI 3사 독립 판결 → 시민 투표 → 최종 판결
- AI: GPT(논리구조), Gemini(균형), Claude(맥락)
