상태 전이 조건

From To Mode 트리거 조건
waiting arguing text duo: B측 사용자가 초대 링크를 통해 참여 / solo: 생성 시 즉시 arguing
waiting chatting chat 양측 사이드 선택 완료 + 방장 시작 클릭 → Socket.io start-chat 이벤트
arguing judging text 양측 주장 입력 완료 (A측 + B측 모두 제출)
chatting judging chat 타이머 15분 만료 OR 상대 이탈(2분 대기 후) → Socket.io chat-ended → 자동 판결 트리거
judging voting all AI 3사 판결 완료 → verdicts 저장 + vote_deadline 설정
voting completed all vote_deadline 도달 → finalizeVerdict() 실행 → 복합 판결 확정
