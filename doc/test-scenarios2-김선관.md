### 알림

#### TC-027: 주장 제출 알림
- 사전조건: 논쟁 진행 중
- 절차:

#### TC-028: 판결 완료 알림
- 사전조건: 판결 진행 완료
- 절차:
  1. 알림 벨 클릭
- 기대결과: "AI 판결이 완료되었습니다" 알림 표시 + 클릭 시 판결 페이지 이동

#### TC-029: 댓글 알림
- 사전조건: 내 논쟁에 타인이 댓글 작성
- 절차:
  1. 알림 벨 확인
- 기대결과: "{닉네임}님이 의견을 남겼습니다" 알림

#### TC-031: 전체 읽음 처리
- 사전조건: 읽지 않은 알림 다수
