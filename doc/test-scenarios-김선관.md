## P0 — 핵심 기능 (반드시 통과)

### 실시간 논쟁 (3v3 채팅 모드)

#### TC-070: 실시간 로비 생성 및 입장
- 사전조건: 실시간 논쟁 모드로 생성 완료
- 절차:
  1. 로비 페이지(`/debate/:debateId/lobby`) 접속
  2. 3v3 아바타 슬롯 UI 확인
- 기대결과: A측/B측 각 3개 슬롯 표시 + 찬성(초록)/반대(빨강) 구분
- 결과: ☐ PASS

#### TC-071: 로비 진영 선택
- 사전조건: TC-070 로비 입장
- 절차:
  1. 빈 슬롯 클릭하여 진영 참여
  2. 같은 슬롯 재클릭하여 선택 해제
- 기대결과: 슬롯에 내 아바타 표시 + 재클릭 시 해제
- 결과: ☐ FAIL (재클릭 해제 불필요 - 구현X)

#### TC-072: 로비 실시간 Presence
- 사전조건: 2명 이상 로비 입장
- 절차:
  1. 다른 유저 로비 입장/퇴장
- 기대결과: Supabase Presence로 실시간 인원 반영 + 입장/퇴장 시스템 메시지
- 결과: ☐ PASS

#### TC-073: 로비 채팅
- 사전조건: 로비 입장
- 절차:
  1. 로비 채팅 입력 + 전송
- 기대결과: Broadcast 채널로 실시간 채팅 표시
- 결과: ☐ FAIL (로비 채팅 미구현 - 구현X)

#### TC-074: 로비 준비 완료 + 게임 시작
- 사전조건: 양측 최소 1명 이상 참여
- 절차:
  1. 양측 "준비완료" 상태 확인
  2. 방장이 "시작" 버튼 클릭
- 기대결과: 방장만 시작 가능 + 양측 준비 시 시작 활성화 → 채팅방 이동
- 결과: ☐ PASS

#### TC-075: 실시간 채팅 기본 플로우
- 사전조건: 게임 시작, ChatRoom 입장
- 절차:
  1. 메시지 입력 (200자 제한) → 전송
  2. 상대방 메시지 수신 확인
  3. 타이핑 인디케이터 확인
- 기대결과: Socket.io 실시간 메시지 송수신 + 1초 쿨다운 + 타이핑 표시
- 결과: ☐ PASS

#### TC-076: 채팅 타이머 및 종료
- 사전조건: 실시간 채팅 진행 중
- 절차:
  1. 15분 타이머 카운트다운 확인
  2. 타이머 종료 또는 "채팅 종료" 버튼 클릭
- 기대결과: 타이머 표시 + 종료 시 판결 플로우 트리거
- 결과: ☐ PASS

#### TC-077: 시민 관전 + 실시간 투표
- 사전조건: 제3자 계정, 실시간 논쟁 진행 중
- 절차:
  1. 채팅방 접속 (관전 모드)
  2. A측 또는 B측 투표 버튼 클릭
- 기대결과: 읽기 전용 채팅 보기 + 실시간 투표 가능 + 투표 현황 표시

#### TC-078: 채팅 중 상대방 이탈 처리
- 사전조건: 실시간 채팅 진행 중

#### TC-079: 채팅 로그 판결 반영
