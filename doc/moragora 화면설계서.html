화면 설명

앱 진입 첫 화면. Supabase Auth를 통한 카카오/구글 OAuth 소셜 로그인만 제공. 자체 회원가입 없이 소셜 계정으로 즉시 가입+로그인 처리.

영역컴포넌트상세
히어로브랜드Navy 그라디언트 + ⚖️ + 로고
카카오Button#FEE500 배경, supabase.auth.signInWithOAuth({provider:'kakao'})
GoogleButtonwhite 배경 + 테두리, supabase.auth.signInWithOAuth({provider:'google'})
약관 안내Text이용약관 및 개인정보처리방침 동의 안내

인터랙션

  • 소셜 버튼 클릭 → OAuth 팝업/리다이렉트 → 로그인 성공 → 홈 피드
  • 최초 로그인 시 자동 회원가입 (Supabase Auth 자동 처리)
  • OAuth 실패 시 toast 에러 메시지
