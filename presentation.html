05 핵심 기능 Key Features 1/2
📝
3단계 논쟁 등록 위자드
Step 1 — 주제 입력 + 모드 선택 (듀오/솔로/채팅)

실시간 채팅 논쟁
Socket.io 기반 실시간 메시지 교환
• 15분 타이머 + 20회 메시지 제한
• 200자/메시지, 1초 쿨다운
• A(초록)/B(빨강) 사이드 컬러
• 채팅 로그 → AI 판결 컨텍스트로 전달

• 5개 항목(논리/근거/설득력/일관성/표현) × 20점
• 총 100점 만점 세부 채점
• 판결문 + 신뢰도(0.5~1.0) 포함
• 실패 시 Grok 자동 폴백
🗳
시민 투표 + 복합 판결
AI 판결 후 24시간 투표 기간
• AI 75% + 시민 25% 가중치
• 30표 이상 달성 시 시민 점수 반영
• 투표 마감 시 자동 최종 판결 확정
• 크론 작업으로 자동 마감 처리

AI 편향 이슈
B측 편향 완전 해소 미완 — 주장 순서 랜덤화만으로는 한계. 프롬프트 엔지니어링 추가 최적화 필요
서버 안정성
Render 무료 플랜의 15분 슬립 문제 — 첫 접속 시 30~60초 지연. 유료 플랜 전환 또는 서버리스 전환 필요
비용 구조
판결 건당 ~185원, 오늘의 논쟁 포함 시 일 1,200원+. 사용자 증가 시 비용 급증 우려
테스트 커버리지
자동화된 테스트 부재 — 수동 QA에 의존. E2E/단위 테스트 도입 필요
🟢 개선 방향
프롬프트 고도화
A/B 순서 완전 랜덤화 + "위치 편향 무시" 지시 강화 + 판결 일관성 테스트 자동화
인프라 개선
Render → AWS Lambda 또는 Railway 전환으로 Cold Start 해소. CDN 캐싱 최적화
